# Pet Registration System

A team-based Python application for processing pet registration forms.

## Project Structure

- `Bugatti/` - Contains all Python scripts
- `team_config.py` - Configuration for team member paths
- `run_script.bat` - Easy launcher for scripts

## Setup

1. Clone this repository
2. Create an Environment folder with a Python virtual environment
3. Install required packages: `pip install -r requirements.txt`
4. Create a Team Members folder with subfolders for each team member
5. Run scripts using `run_script.bat`

## Required Folder Structure

SPMS_Pet_Registration/
├── Bugatti/            # Scripts folder
├── Environment/        # Python environment
├── Team Members/       # Team member folders
│   ├── [Member1]/
│   │   ├── PetForms/   # Input files
│   │   └── Uploads/    # Output files
│   └── [Member2]/
│       ├── PetForms/
│       └── Uploads/
└── run_script.bat      # Script launcher
